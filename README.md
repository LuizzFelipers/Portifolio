Blue Chips Vs Startups (Financas.py):
- Objetivo é analisar como os indicadores macroeconômicos impactam em pequenas e grandes empresas. Os indicadores são IPCA, SELIC, PIB e o Câmbio.
- Utilizei Pandas para manipulação de dados, Plotly para os Gráficos, API'S do Yahool Finance e Banco Central e o Streamlit para criar o dashboard web.
 
Projeto de Inadimplência (Inadimplência.py):
- Objetivo é prever, através de uma arquivo excel de um setor de Contas a Receber, riscos de inadimplências de clientes.
- Pandas e Openpyxl para tratamento e manipulação da base de dados Excel, Plotly para os gráficos e Streamlit para criar o dashboard.

Detran
- O objetivo é analisar a base de dados do detrans e responder algumas perguntas como: Dia da semana com mais acidentes, rodovia mais perigosa,etc.
- Pandas para manipular os dados, plotly para gráficos, streamlit para criar o dashboard web e requests para fazer uma requisição via API.
