Blue Chips Vs Startups
- Objetivo é analisar como os indicadores macroeconômicos impactam em pequenas e grandes empresas. Os indicadores são IPCA, SELIC, PIB e o Câmbio.
- Utilizei Pandas para manipulação de dados, Plotly para os Gráficos, API'S do Yahool Finance e Banco Central e o Streamlit para criar o dashboard web.
